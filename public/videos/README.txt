Ciao Josue!

In questa cartella puoi caricare i video reali del tuo lavoro, OPPURE puoi usare dei link esterni direttamente!

METODO 1: USARE LINK ESTERNI (YouTube, Vimeo, Google Drive, ecc.) - Consigliato!
Se vuoi usare dei video già presenti su internet (o se non riesci a caricare file pesanti qui):
1. Apri il file `/src/portfolioData.ts` usando l'esploratore file a sinistra.
2. Trova la sezione `reels`.
3. Sostituisci il valore di `videoUrl` con il link del tuo video. Ad esempio:
   - Un link di Google Drive pubblico: `"https://drive.google.com/file/d/CODICE_FILE/view?usp=sharing"`
   - Un link standard di YouTube: `"https://www.youtube.com/watch?v=CODICE_VIDEO"`
   - Un link standard di Vimeo: `"https://vimeo.com/CODICE_VIDEO"`
   - Un link diretto a un file video online (che termina in `.mp4`)
4. Salva il file. Il telefono nel portfolio riprodurrà automaticamente il video dal link!

METODO 2: CARICARE I FILE MP4 LOCALI
Se vuoi caricare i file direttamente dentro l'app, trascinali (Drag & Drop) in questa cartella `/public/videos/` nominando i file in minuscolo così:
- `reel1.mp4` (il video delle interviste / corsi ITS AMMI)
- `reel2.mp4` (il video sulle materie pratiche e Digital Marketing)
- `reel3.mp4` (il video della mostra Guess / Marilyn Monroe)
- `reel4.mp4` (il video del calcio balilla / Scegli AMMI)

L'applicazione gestirà automaticamente entrambi i metodi in modo perfetto!
